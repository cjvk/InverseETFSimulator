# InverseETFSimulator

## Steps to clone

1. Go to https://github.com/cjvk/InverseETFSimulator
2. On the right panel, find 'HTTPS clone URL', and click the clipboard icon next to it.
3. `$ git clone <clipboard_contents>`
  * `$ git clone https://github.com/cjvk/InverseETFSimulator.git`
